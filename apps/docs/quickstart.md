# Quickstart

## 1. Install dependencies

```bash
pnpm install
```

## 2. Start docs locally

```bash
pnpm docs:dev
```

## 3. Try the CLI MVP

```bash
pnpm ckit:init
pnpm ckit:generate
pnpm ckit:check
```
